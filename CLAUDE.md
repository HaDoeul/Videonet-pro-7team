# VideoNet Pro - Claude 작업 규칙

## 절대 금지 사항

### 1. 목업/테스트/임시 데이터 절대 금지
- ❌ 목업 데이터 생성 금지
- ❌ 간단한 테스트용 구현 금지
- ❌ 임시 방편 해결 금지
- ❌ 가짜 데이터 생성 금지
- ✅ 실제 동작하는 완전한 구현만 허용

### 2. 근본적인 문제 해결
- 문제의 근본 원인을 파악하고 완전히 해결
- 임시방편이 아닌 영구적인 솔루션 제공
- 모든 엣지 케이스 고려

### 3. 프로덕션 품질
- 실제 서비스 가능한 코드만 작성
- 에러 처리 철저히
- 로깅 및 모니터링 구현
- 보안 고려

## 프로젝트 요구사항

### VideoNet Pro
**20205146 한림대학교 콘텐츠IT 김재형, AI+X 프로젝트 과제**
**videonet C 프로젝트를 기반으로 만들었음**

### 핵심 기능
1. **P2P 파일 전송**
   - 무손실 전송 (해시 검증)
   - 최소 대역폭 사용
   - 화상채팅 품질 저하 없음
   - 동일한 파일 전송 보장

2. **WebRTC 화상채팅**
   - 다중 사용자 지원
   - 실시간 동기화
   - 안정적인 연결

3. **동영상 분석**
   - 슬라이싱 기반 요약
   - GPT API 인물 인식
   - 텍스트 데이터로 변환 (토큰 절약)

### 기술 스택
- Frontend: 7700 포트

- systemd로 자동 시작
- 방화벽 설정 완료
